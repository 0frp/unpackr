## Unpackr Release

### Official assets
- `Unpackr-Setup-<version>.exe`
- `SHA256SUMS.txt`

This project intentionally ships a single installer artifact for public users.

### Install
1. Download `Unpackr-Setup-<version>.exe`
2. Run installer
3. Launch from desktop/start menu shortcut

### Verify
Compare installer SHA256 with `SHA256SUMS.txt` before installing.
